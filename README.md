
# Colegio Humboldt - Frontend

Este proyecto es el frontend de la página web del Colegio Humboldt, creado con React y TypeScript.

## Estructura principal

- `src/components`: Componentes reutilizables
- `src/pages`: Páginas principales (ej. LandingPage)
- `src/assets`: Imágenes y otros recursos
- `src/styles`: Hojas de estilo globales o compartidas

In the project directory, you can run:


### `npm start`
Inicia la aplicación en modo desarrollo.
Abre [http://localhost:3000](http://localhost:3000) para verla en el navegador.


### `npm test`
Ejecuta los tests en modo interactivo.


### `npm run build`
Genera una versión optimizada para producción en la carpeta `build`.


### `npm run lint`
Ejecuta ESLint para revisar la calidad del código.

### `npm run format`
Ejecuta Prettier para formatear el código automáticamente.


## Notas

- Asegúrate de tener Node.js 18 o superior para evitar advertencias de compatibilidad.
- ESLint y Prettier están instalados para mantener la calidad y el formato del código.

¡Feliz desarrollo!
